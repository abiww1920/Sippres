1. PENDAHULUAN
================================================================================

SIPPRES (Sistem Informasi Pelanggaran dan Prestasi Siswa) adalah aplikasi 
berbasis web yang dirancang untuk membantu sekolah dalam mengelola data 
pelanggaran dan prestasi siswa secara digital dan terintegrasi.

TUJUAN APLIKASI:
- Mencatat dan memantau pelanggaran siswa
- Mencatat dan mengelola prestasi siswa
- Memberikan sanksi otomatis berdasarkan poin pelanggaran
- Memfasilitasi bimbingan konseling
- Menghasilkan laporan komprehensif
- Memberikan notifikasi kepada orang tua

PENGGUNA APLIKASI:
1. Admin - Mengelola seluruh data master sistem
2. Kesiswaan - Mengelola pelanggaran, prestasi, dan verifikasi
3. Guru - Mencatat pelanggaran siswa
4. Wali Kelas - Memantau siswa di kelasnya
5. Konselor - Melakukan bimbingan konseling
6. Kepala Sekolah - Monitoring dan laporan
7. Siswa - Melihat data pelanggaran dan prestasi pribadi
8. Orang Tua - Memantau perkembangan anak

2. CARA MENGAKSES APLIKASI
================================================================================

PERSYARATAN SISTEM:
- Browser: Google Chrome, Mozilla Firefox, Microsoft Edge (versi terbaru)
- Koneksi Internet
- Resolusi layar minimal 1366x768

CARA AKSES:
1. Buka browser
2. Ketik URL: http://localhost/UjikomAbi (untuk lokal)
   atau sesuai domain yang diberikan sekolah
3. Halaman login akan muncul

4. PANDUAN PER ROLE USER

--------------------------------------------------------------------------------
4.1 ADMIN
--------------------------------------------------------------------------------

MENU YANG TERSEDIA:
1. Dashboard - Statistik keseluruhan sistem
2. Data Guru - Kelola data guru
3. Data Siswa - Kelola data siswa
4. Data Kelas - Kelola data kelas
5. Data Jurusan - Kelola data jurusan
6. Data User - Kelola akun pengguna
7. Kategori Pelanggaran - Kelola jenis pelanggaran
8. Data Pelanggaran - Lihat semua pelanggaran
9. Data Prestasi - Lihat semua prestasi
10. Data Sanksi - Kelola sanksi
11. Tahun Ajaran - Kelola tahun ajaran
12. Laporan - Generate laporan

CARA MENAMBAH DATA GURU:
1. Klik menu "Data Guru"
2. Klik tombol "Tambah Guru"
3. Isi form:
   - NIP
   - Nama Guru
   - Email
   - No. Telepon
   - Mata Pelajaran
   - Status
4. Klik "Simpan"

CARA MENAMBAH DATA SISWA:
1. Klik menu "Data Siswa"
2. Klik tombol "Tambah Siswa"
3. Isi form:
   - NIS
   - Nama Siswa
   - Email
   - Kelas
   - Jurusan
   - Jenis Kelamin
   - Alamat
   - No. HP Siswa
   - Nama Orang Tua
   - No. HP Orang Tua
   - Upload Foto (opsional)
4. Klik "Simpan"

CARA MENGELOLA KELAS:
1. Klik menu "Data Kelas"
2. Untuk tambah: Klik "Tambah Kelas"
3. Isi: Nama Kelas, Tingkat, Wali Kelas
4. Klik "Simpan"

CARA MENGELOLA TAHUN AJARAN:
1. Klik menu "Tahun Ajaran"
2. Klik "Tambah Tahun Ajaran"
3. Isi: Tahun Ajaran (contoh: 2024/2025)
4. Centang "Aktif" jika tahun ajaran sedang berjalan
5. Klik "Simpan"

--------------------------------------------------------------------------------
4.2 KESISWAAN
--------------------------------------------------------------------------------

MENU YANG TERSEDIA:
1. Dashboard - Statistik pelanggaran dan prestasi
2. Data Pelanggaran - Kelola pelanggaran
3. Data Prestasi - Kelola prestasi
4. Verifikasi - Verifikasi data pelanggaran/prestasi
5. Monitoring - Pantau siswa bermasalah
6. Laporan - Generate laporan

CARA MENCATAT PELANGGARAN:
1. Klik menu "Data Pelanggaran"
2. Klik "Tambah Pelanggaran"
3. Isi form:
   - Pilih Siswa (cari berdasarkan nama/NIS)
   - Pilih Jenis Pelanggaran
   - Poin akan otomatis terisi
   - Isi Keterangan
   - Upload Foto Bukti (jika ada)
4. Klik "Simpan"
5. Status: "Menunggu Verifikasi"

CARA MENCATAT PRESTASI:
1. Klik menu "Data Prestasi"
2. Klik "Tambah Prestasi"
3. Isi form:
   - Pilih Siswa
   - Pilih Jenis Prestasi
   - Tingkat (Sekolah/Kecamatan/Kabupaten/Provinsi/Nasional)
   - Juara (1/2/3/Harapan)
   - Tanggal Prestasi
   - Keterangan
   - Upload Sertifikat (opsional)
4. Klik "Simpan"

CARA VERIFIKASI DATA:
1. Klik menu "Verifikasi"
2. Akan muncul daftar data yang menunggu verifikasi
3. Klik "Detail" pada data yang ingin diverifikasi
4. Periksa data dengan teliti
5. Pilih:
   - "Verifikasi" jika data benar
   - "Tolak" jika data tidak sesuai (isi alasan penolakan)
6. Klik "Submit"

CARA MONITORING SISWA:
1. Klik menu "Monitoring"
2. Sistem menampilkan siswa dengan poin pelanggaran tinggi
3. Klik nama siswa untuk melihat detail
4. Lihat riwayat pelanggaran dan sanksi
5. Bisa cetak laporan per siswa

CARA MEMBUAT LAPORAN:
1. Klik menu "Laporan"
2. Pilih jenis laporan:
   - Laporan Pelanggaran
   - Laporan Prestasi
   - Laporan Per Siswa
   - Rekap Bulanan
3. Filter berdasarkan:
   - Tanggal
   - Kelas
   - Jenis Pelanggaran/Prestasi
4. Klik "Generate Laporan"
5. Pilih format: PDF atau Excel
6. Klik "Download"

5. FITUR-FITUR UTAMA
================================================================================

--------------------------------------------------------------------------------
5.1 SISTEM POIN
--------------------------------------------------------------------------------

POIN PELANGGARAN:
- Ringan: 5-15 poin
- Sedang: 20-30 poin
- Berat: 50-75 poin
- Sangat Berat: 100 poin

POIN PRESTASI:
- Sekolah: 5-10 poin
- Kecamatan: 15-20 poin
- Kabupaten: 25-30 poin
- Provinsi: 35-40 poin
- Nasional: 45-50 poin

POIN BERSIH:
Poin Bersih = Total Poin Prestasi - Total Poin Pelanggaran

--------------------------------------------------------------------------------
5.2 SISTEM SANKSI OTOMATIS
--------------------------------------------------------------------------------

SANKSI BERDASARKAN AKUMULASI POIN:
- 25-49 poin: Teguran tertulis + Panggilan orang tua
- 50-74 poin: Skorsing 1 hari + Bimbingan konseling
- 75-99 poin: Skorsing 3 hari + Surat pernyataan
- 100+ poin: Skorsing 1 minggu + Pertimbangan dikeluarkan

PELAKSANAAN SANKSI:
1. Sanksi otomatis dibuat saat pelanggaran diverifikasi
2. Wali kelas memantau pelaksanaan
3. Status sanksi diupdate sesuai pelaksanaan
4. Orang tua mendapat notifikasi

--------------------------------------------------------------------------------
5.3 VERIFIKASI DATA
--------------------------------------------------------------------------------

PROSES VERIFIKASI:
1. Guru/Wali Kelas input pelanggaran
2. Status: "Menunggu Verifikasi"
3. Kesiswaan melakukan verifikasi
4. Jika disetujui: Status "Diverifikasi", sanksi otomatis dibuat
5. Jika ditolak: Status "Ditolak", beri alasan

TUJUAN VERIFIKASI:
- Memastikan keakuratan data
- Mencegah kesalahan input
- Validasi bukti pelanggaran

6. FAQ (FREQUENTLY ASKED QUESTIONS)
================================================================================

Q: Bagaimana cara reset password?
A: Hubungi administrator sekolah untuk reset password.

Q: Apakah bisa edit data pelanggaran yang sudah diverifikasi?
A: Tidak. Data yang sudah diverifikasi tidak bisa diedit. Hubungi kesiswaan
   jika ada kesalahan.

Q: Bagaimana cara upload foto bukti pelanggaran?
A: Saat input pelanggaran, klik tombol "Choose File" di field Foto Bukti,
   pilih file gambar (JPG/PNG), maksimal 2MB.

Q: Apakah prestasi bisa mengurangi poin pelanggaran?
A: Ya. Sistem menghitung Poin Bersih = Poin Prestasi - Poin Pelanggaran.

Q: Bagaimana cara melihat siswa dengan pelanggaran terbanyak?
A: Klik menu Monitoring, sistem akan menampilkan siswa dengan poin tertinggi.

Q: Apakah orang tua bisa login ke sistem?
A: Ya. Orang tua mendapat akun untuk memantau perkembangan anak.

Q: Bagaimana cara mengganti tahun ajaran?
A: Admin masuk ke menu Tahun Ajaran, tambah tahun ajaran baru, dan set
   sebagai aktif.

Q: Apakah bisa export data ke Excel?
A: Ya. Di menu Laporan, pilih "Export Excel".

Q: Bagaimana cara menghubungi konselor?
A: Lihat data konselor di menu Data Guru atau hubungi bagian kesiswaan.

Q: Apakah ada backup data?
A: Ya. Sistem melakukan backup otomatis. Hubungi admin untuk restore data.

Q: Bagaimana cara mencetak laporan?
A: Generate laporan dalam format PDF, lalu print dari browser.

Q: Apakah bisa filter data berdasarkan kelas?
A: Ya. Hampir semua menu memiliki fitur filter berdasarkan kelas.

Q: Bagaimana cara menambah jenis pelanggaran baru?
A: Admin masuk ke menu Kategori Pelanggaran, klik Tambah, isi data.

Q: Apakah siswa bisa melihat data siswa lain?
A: Tidak. Siswa hanya bisa melihat data pribadi mereka sendiri.

Q: Bagaimana cara update foto profil siswa?
A: Admin atau Kesiswaan edit data siswa, upload foto baru.
