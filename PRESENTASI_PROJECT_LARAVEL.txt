================================================================================
1. INFORMASI PROJECT
================================================================================

Nama Aplikasi    : SIPPRES (Sistem Informasi Pelanggaran dan Prestasi Siswa)
Framework        : Laravel 12.0
PHP Version      : 8.2+
Database         : MySQL (sippres)
Template Admin   : Flexy Bootstrap Lite
PDF Generator    : DomPDF (barryvdh/laravel-dompdf)

Deskripsi:
Sistem informasi berbasis web untuk mengelola data pelanggaran dan prestasi 
siswa di sekolah dengan fitur multi-role, notifikasi otomatis, monitoring 
sanksi, bimbingan konseling, dan laporan lengkap.

8. CARA INSTALASI
================================================================================

REQUIREMENTS:
- PHP >= 8.2
- MySQL/MariaDB
- Composer
- Node.js & NPM
- XAMPP/WAMP/LARAGON

LANGKAH INSTALASI:

1. Clone/Download Project
   cd c:\xampp\htdocs\UjikomAbi

2. Install Dependencies
   composer install
   npm install

3. Setup Environment
   - Copy .env.example ke .env
   - Sesuaikan konfigurasi database:
     DB_DATABASE=sippres
     DB_USERNAME=root
     DB_PASSWORD=

4. Generate Application Key
   php artisan key:generate

5. Setup Database
   - Buat database 'sippres' di MySQL
   - Import file: public/sippres.sql
   ATAU
   - Run migrations: php artisan migrate
   - Run seeders: php artisan db:seed

6. Setup Storage
   php artisan storage:link

7. Build Assets
   npm run build

8. Run Application
   php artisan serve
   
9. Akses Aplikasi
   http://localhost:8000

14. TROUBLESHOOTING
================================================================================

MASALAH UMUM & SOLUSI:

1. Error 500 - Internal Server Error
   Solusi: 
   - Cek file .env
   - php artisan config:clear
   - php artisan cache:clear

2. Database Connection Error
   Solusi:
   - Pastikan MySQL running
   - Cek kredensial di .env
   - Pastikan database 'sippres' sudah dibuat

3. Upload File Gagal
   Solusi:
   - Cek permission folder public/uploads
   - php artisan storage:link
   - Cek php.ini: upload_max_filesize & post_max_size

4. Route Not Found
   Solusi:
   - php artisan route:clear
   - php artisan route:cache

5. View Not Found
   Solusi:
   - php artisan view:clear
   - Cek nama file blade

6. Email Tidak Terkirim
   Solusi:
   - Cek konfigurasi MAIL di .env
   - php artisan queue:work (untuk queue)
